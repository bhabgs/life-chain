# 生命链 - 页面与功能目录结构

> 基于产品需求文档梳理的完整功能架构
>
> **项目分端**：Mobile（移动端APP）、MiniApp（小程序）、Web-Admin（管理后台）、Server（服务端）

---

## 📋 端适配策略说明

| 端 | 定位 | 功能覆盖 | 适用场景 |
|---|------|---------|---------|
| **移动APP** | 主力产品 | 100%完整功能 | 深度使用、完整陪伴 |
| **小程序** | 轻量入口 | 30-40%核心功能 | 快速记录、轻量查看 |
| **Web管理后台** | 运营管理 | 管理功能 | 后台管理、数据分析 |

### 小程序页面限制说明

| 限制类型 | 说明 | 解决方案 |
|---------|------|---------|
| **语音录制** | 最长60秒 | 引导用户使用APP录制长语音 |
| **本地存储** | 最多10MB | 只缓存最近记忆，其余云端获取 |
| **视频录制** | 不支持 | 引导用户使用APP |
| **高级功能** | 不支持VR、硬件联动等 | 引导用户下载APP |
| **复杂配置** | 简化配置界面 | 高级设置在APP完成 |

## 📝 备注说明

1. **技术栈建议**
   - 移动端：React Native（跨平台）或 Flutter
   - Web-Admin：React + Ant Design 或 Vue + Element Plus
   - 服务端：Node.js (NestJS) 或 Java (Spring Boot)

2. **数据库设计**
   - 主数据库：PostgreSQL（结构化数据）
   - 缓存：Redis
   - 文件存储：OSS/S3
   - 向量数据库：用于记忆检索和AI功能

3. **AI能力需求**
   - 大语言模型：对话生成、情绪识别
   - 语音识别：ASR服务
   - 情感分析：文本/语音情绪分析
   - 知识图谱：人物关系网络

4. **安全与隐私**
   - 数据加密存储
   - 敏感信息脱敏
   - 权限分级控制
   - 合规审计日志
